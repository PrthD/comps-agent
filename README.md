# KV Comps Agent

An explainable, conservative comps-valuation agent for lenders — built for the KV Capital
AI Engineer hackathon. It mirrors an appraiser's sales-comparison workflow: retrieve plausible
recent comps, flag $/sqft outliers, and produce a *defensible, conservative* value with a range,
confidence, and plain-English rationale — not a black-box point estimate.
